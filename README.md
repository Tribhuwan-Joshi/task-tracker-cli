# CLI Todo App

A simple CLI-based todo app built using Node.js, with tasks stored in a JSON file using the `fs` module.

## Features

- Add, update, delete, mark tasks as `todo`, `in-progress`, or `done`
- List tasks or filter by status

## Usage

Use `npm run task-cli` followed by the desired command.

### Add a Task
```bash
npm run task-cli add "Task description"
```
